# ThRACE
a satellite project meant to serve as essentially a "capstone" project for all the stuff i have learned about rocketry and space.

This repository is meant to be used as a reference. This is the first satellite I have ever made, and so far my knowledge only goes up to level 2 high power rockets. This will be updated as progress is made, and hopefully, it will be flown.

# IMPORTANT BACKGROUND
I am starting this project as a sophmore in high school. I have been into space for many years and I have always wanted to put this thing there.

# TLDR ON WHAT THIS PROJECT IS
basically its a satellite that goes into a 45º inclination orbit (just LEO), measures radiation and temperature, sends that data back down to the groundstation, and helps validate pre-flight simulations that were used to decide on the shielding. pretty simple, I think.

have fun exploring what I have made so far! (u can tell ive never made something like this before)
